# Endoscopy ML Project

This repository is being organized as a complete end-to-end ML project.

## Intended workflow

1. Build the dataset manifest
2. Perform exploratory data analysis
3. Create the dataset and preprocessing module
4. Build the model module
5. Add training and evaluation scripts
6. Add MLOps tracking for metrics and artifacts
7. Add inference and deployment hooks later

## Planned structure

- `build_dataset.py` — dataset manifest builder
- `dataset/` — split CSV files and summary metadata
- `src/` — data, model, training, evaluation, and inference modules
- `mlops/` — experiment tracking and run metadata
- `EDA/` — exploratory analysis notebook
- `config/` — configuration files
- `artifacts/` — saved outputs and experiment artifacts

## Current status

The project skeleton is in place. Each module is intentionally left as a placeholder so the workflow can be built step by step.
